作品名称：基于YOLO蒸馏与轻量化的智能垃圾识别算法
本项目针对现有垃圾检测算法在边缘设备部署中存在计算量大、实时性不足的问题，提出了一种基于YOLO蒸馏与轻量化优化的智能垃圾识别算法。首先在Kaggle垃圾分类数据集上训练YOLO11m教师模型，通过生成伪标签增强未标注样本，从而构建高质量蒸馏数据集。随后采用YOLO11n作为学生模型进行知识蒸馏学习，有效继承教师特征表达能力，同时大幅降低参数量与计算复杂度。最后结合结构剪枝（pruning）与量化（quantization）优化，实现模型在边缘设备上的实时部署与高精度识别。实验结果表明，该方案在保证mAP精度接近教师模型的同时，模型体积缩减约68%，推理速度提升约2.3倍，显著提高了边缘部署的可行性与能效表现。该算法具有较高的创新性与实用性，可广泛应用于校园、社区、城市清洁等智能环境感知场景
